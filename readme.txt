Разбор графических файлов PNG.

На вход программа принимает изображение и название чанка, разбор которого вы хотите видеть,
если вы хотите увидеть разбор всех чанков, которые есть в вашем изображении,
то вместо чанка нужно передать команду "all".

Фаил create_image.py выводит hex - запись изображения.
Фаил analysis.py вывод перечь всех чанков, которые содержатся в изображении и 
разбор указанного вами чанка.
Фаил visualization.py вывод само изображение и результат файла analysis.py.

*Для запуска файла visualization.py необходимо скачать библиотеку PyQt.
